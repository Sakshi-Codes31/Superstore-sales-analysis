# Superstore Sales Analysis

## Objective
Analyze 9,800+ sales orders to identify top-performing regions, categories, and customer segments using SQL and Tableau.

## Tools Used
- SQL (SQLLite Online)
- Tableau Public
- Excel / OpenOffice

## Dataset
- Source: Kaggle — Superstore Sales Dataset by Rohit Sahoo
- Size: 9,800 rows, 18 columns
- Period: 2014–2018

## Key Business Insights
- West region generates the highest sales ($710,219)
- Technology is the top-selling category ($827,455)
- Consumer segment contributes 51% of total sales ($1,148,060)
- California is the #1 state by sales ($446,306)
- New York is the #2 state by sales ($306,361)

## SQL Queries
See queries.sql for all queries including:
- Sales by Region
- Sales by Category
- Sales by Customer Segment
- Top 10 States by Sales
